    Getting Started

    These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

    Installation

    Windows

    • Download the latest release .exe file.

    • Run the exe file and install GuptaK.

    • Launch program!

    x64 only! Use the .shy for x32 systems!

    Mac OS

    • Download the latest version of Shoes 3.

    • Download the latest release .shy file for OSX.

    • Launch the .shy using Shoes 3.

    Linux

    • Download the latest version of Shoes 3.

    • Download the latest release .shy file for Linux.

    • Launch the .shy using Shoes 3.
